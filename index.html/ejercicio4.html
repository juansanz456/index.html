Ejercicio N°4

Himno del sena

CORO
Estudiantes del SENA adelante Por Colombia luchad con amor Con el animo noble y radiante Transformémosla en mundo mejor
I
De la patria el futuro destino, en las manos del joven está, el trabajo es seguro camino, que el progreso a Colombia dará.
II
En la forja del SENA se forman, hombres libres que anhelan triunfar, con la ciencia y la técnica unidas, nuevos rumbos de paz trazarán.
III
Hoy la patria nos grita sentida, ¡estudiantes del SENA triunfad! solo así lograréis en la vida, más justicia, mayor libertad.
IV
Avancemos con fuerza guerrera, ¡estudiantes con firme tesón! que la patria en nosotros espera, su pacífica revolución.

etiquetas

img

La etiqueta img en HTML se utiliza para insertar imágenes en una página web. Es una etiqueta autocontenida (es decir, no tiene una etiqueta de cierre) y requiere al menos el atributo src para especificar la ubicación de la imagen.

dialog

La etiqueta dialog en HTML se utiliza para crear cuadros de diálogo modales o no modales en una página web.

article

La etiqueta article en HTML se utiliza para definir contenido independiente y autocontenido que podría distribuirse de manera independiente o reutilizarse.

meter

Muestra un valor dentro de un rango.

figcaption

La etiqueta figcaption en HTML se utiliza para proporcionar una leyenda o descripción para el contenido de una figura.

time

Define una fecha o hora en formato máquina.

details

Crean un widget que muestra o oculta información adicional.

data

La etiqueta data es una etiqueta HTML que se utiliza para asociar un valor con un contenido visible.

section

La etiqueta representa una sección genérica de un documento.

nav

La etiqueta representa una sección de navegación en un documento.
